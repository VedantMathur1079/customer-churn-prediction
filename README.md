# 📊 Customer Churn Prediction

## 📌 Overview
A Machine Learning project that predicts customer churn using historical data. Built using Python and deployed using Streamlit.

---

## 🔥 Features
- Predict customer churn
- Data preprocessing and feature engineering
- Random Forest model
- Interactive Streamlit app

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
